# サービス概要

ニートになるのが怖い、将来どうなるか不安という悩みを持った人に
どういうニートになれるかキャリアと適性を示す
ニートで構成された架空の会社への就活アプリ

# 登場人物

現在ニートで将来が不安な不就労者

# ユーザーが抱える悩み

今の仕事に満足していないがニートになる方が先が見えず退職に踏み切れない、現状の貯金額でどれくらい無職で生活できるか、将来が不安。

# 解決方法

心理テストに答えてもらい、貯金額を入力してもらうことで雇用条件や期間、給料を算出しニートとしての市場価値を図る。
